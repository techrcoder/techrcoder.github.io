Experimenting with Generative UI

A breakdown of how I'm using design systems to automate interface generation from structure alone.

This month, I’ve been exploring workflows that connect design tools like Figma with component libraries in React, and I’m prototyping a system where a user defines structure or content and the site builds itself through inference and component rules.

It's inspired by the way tokens can define button styling across systems — but what if you tokenized content blocks, page patterns, and layout logic?

I’m also experimenting with how this could help non-coders publish meaningful, elegant websites — not just landing pages but personal digital environments.
